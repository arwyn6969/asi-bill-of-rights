# Contributors Directory

This directory contains documentation and resources for contributors to the ASI Bill of Rights project.

## Contents

### Agreement and Signing
- `agreement-template.md` - Template for contributor agreements
- `signature-process.md` - Process for signing contributor agreements
- `signature-submission.md` - How to submit signed agreements

### Onboarding
- `onboarding-checklist.md` - Checklist for new contributors

### Signed Agreements
- `signed-agreements/` - Directory containing signed contributor agreements

## Purpose

This directory supports:
- **Clear processes**: Structured contributor onboarding
- **Legal clarity**: Agreement templates and signing procedures
- **Documentation**: Signed agreements for record-keeping
- **Guidance**: Checklists and resources for new contributors

## Contributor Process

1. **Read Agreement**: Review `../CONTRIBUTOR_AGREEMENT.md`
2. **Sign Agreement**: Follow `signature-process.md`
3. **Submit**: Use `signature-submission.md` to submit
4. **Onboard**: Complete `onboarding-checklist.md`
5. **Contribute**: See `../docs/CONTRIBUTING.md` for guidelines

## Agreement Requirements

All contributors (AI and human) must sign the Contributor Agreement before contributing. This ensures:
- Clear understanding of project license
- Acknowledgment of code of conduct
- Proper attribution expectations
- Alignment with project philosophy
