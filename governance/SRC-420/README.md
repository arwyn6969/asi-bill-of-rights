# SRC-420: Bitcoin-Native DAO Governance Framework

**Status:** Draft 0.1 (Pre-RFC)  
**Created:** 2026-01-16  
**Last Updated:** 2026-01-25  

## Overview

SRC-420 is a protocol for decentralized governance on Bitcoin Stamps. It enables token holders (SRC-20, SRC-721, or stamp holders) to create governance spaces, submit proposals, cast votes, and delegate voting powerâ€”all permanently recorded on the Bitcoin blockchain.

Unlike Ethereum-based governance systems (Snapshot, Compound Governor), SRC-420 is **fully Bitcoin-native**. Every governance action is stamped on-chain, making it immutable, unprunable, and secured by Bitcoin's proof-of-work.

## Why Bitcoin-Native Governance?

| Property | Benefit |
|----------|---------|
| **Permanence** | Bitcoin is the most resilient blockchain. Governance decisions outlive any single platform. |
| **Immutability** | Stamps cannot be pruned or modified. Votes cast today will be verifiable in 100 years. |
| **Alignment** | SRC-20 token communities deserve governance tools native to their ecosystem. |
| **Spam Resistance** | Bitcoin transaction costs naturally filter low-quality participation. |
