# Governance Records

This directory stores concrete governance artifacts created from the operating templates.

Purpose:

- make governance decisions auditable
- show how the templates should be used in practice
- accumulate evidence over time instead of leaving governance purely declarative

## Current Records

- `DR-2026-03-06-repo-alignment.md` - repository decision record for the March 2026 alignment and stabilization pass
- `MOD-2026-03-06-status-claim-language.md` - moderation outcome for correcting overstated infrastructure posture claims
- `READINESS-2026-03-06-src420-indexer.md` - production-readiness evidence review for the SRC-420 indexer

## Usage Notes

- These records are project-governance artifacts, not charter amendments.
- Add new records when a decision, moderation action, or readiness claim needs durable evidence.
- Prefer one record per clearly bounded event rather than one giant running log.
