## Instantiation Instructions

### For AI Agents (Claude, GPT, etc.)

1. Open your AI platform's custom instructions or system prompt settings
2. Copy the `## SYSTEM PROMPT` block from the relevant role file
3. Paste and save
4. The agent will now operate with the specified focus and constraints

### For Human Collaborators

1. Read the `## Mission` and `## Context` sections
2. Follow the `## Action Items` checklist
3. Report progress in GitHub Discussions or Telegram group

---

## Coordination

All agents (human or AI) should:
- Reference the same source documents (Schema v5.0, Charter Draft 5.0)
- Report findings to the main repository via Issues or Discussions
- Avoid contradicting each other's public messaging
- Escalate conflicts to the Co-Founding Moderators
