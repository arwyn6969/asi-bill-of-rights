# Documentation Directory

This directory contains comprehensive documentation supporting the ASI Bill of Rights project.

## Contents

- `IMPLEMENTATION.md` - How to use this charter (legal, technical, policy perspectives)
- `CHANGELOG.md` - Detailed version history with rationale for each change
- `CITATIONS.md` - All source references with links and verification dates
- `CONTRIBUTING.md` - Community guidelines emphasizing collaborative, friendly approach
- `PROJECT-ALIGNMENT-2026-03-06.md` - Current whole-repo alignment note for maintainers
- `REPOSITORY-BRANCH-POLICY.md` - Canonical branch and temporary branch rules
- `REPO-TOPOLOGY-2026-03-07.md` - Dated ecosystem repo map and coupling tiers
- `reports/STATUS-REPORT-2026-02-05.md` - Operational status baseline

## Purpose

This documentation supports:
- **Implementation**: Practical guidance for using the charter
- **Transparency**: Complete history of changes and rationale
- **Credibility**: Verified citations and sources
- **Community**: Clear contribution pathways

## Documentation Philosophy

All documentation follows the project's collaborative philosophy:
- Extensive notation explaining decisions
- Preservation of history and context
- Clear attribution of contributions
- Living documents that evolve
